Implementace

  • Změny od use-case diagramu
    • Oddělili jsme Pána jeskyně a Admina
      (Ne každý admin se hodí na PJ)
    • Tvorbu tažení, dobrodružství a sezení jsme ponechali pouze pánu jeskyně
      (Bylo to v plánu, ale pro správnou funkčnost by muselo být implementováno i nějaké fórum, kde by se např. hledali PJ, atd.)

Instalace

  • Požadavky na instalaci
    • PHP 7.1
    • Nette 2.4
    • Bootstrap 4.0
  • Podle návodu na nette.org se nainstaluje nette (manuálně nebo přes Composer)
  • V archívu jsou pak složky 'app' a 'www', ty stačí přesunout do kořene serveru
  • Následně stačí importovat přibalenou databázi (např. pomocí phpMyAdmin)
  • Server by nyní měl fungovat

Známé problémy

  • Debuger je zapnut, protože když se vypne všechny formuláře hážou 500 (důvod víme, řešení však ne)
    Z toho důvodu, pokud vyskočí výjimka, objeví se výpis z debugeru
  • Nefunguje hra (spuštění sezení v menu Pán Jeskyně)
    Lze ji spustit, ale nejde ji hrát (nelze tedy ani zabít postavu)
  • Admin nemůže tvořit ani mazat, pouze editovat
  • Není možnost vytisknout hromadnou statistiku
  • Nefunguje úprava datumu
